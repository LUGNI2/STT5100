Numerical optimization

``` r
minuslogL = function(theta){
  mu=theta[1]
  sigma=theta[2]
  logL=sum( log(dnorm(Y,mu,sigma) ))
  return(-logL)
}
optim(par = c(150,20),minuslogL,hessian=FALSE)$par
```
