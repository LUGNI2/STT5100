``` r
(Bhat = cor(X1,Y) * sd(Y)/sd(X1))
```

    ## [1] 77.28258

``` r
(Ahat = mean(Y) - Bhat*mean(X1))
```

    ## [1] -288.6079

``` r
plot(X1,Y)
abline(v=mean(X1),col="red",lty=2)
abline(h=mean(Y),col="red",lty=2)

```

``` r
Y_hat = Ahat + Bhat*X1
```
